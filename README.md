# BCA 4th Semester - Java Programming

This repository contains the Java programming assignments and practicals for the **BCA 4th Semester**. All programs are developed and tested using **VS Code**.

## 📂 Repository Contents
* Java source code files (`.java`)

## 🛠️ Requirements
* **Editor:** Visual Studio Code (VS Code)
* **Extension:** Extension Pack for Java (by Microsoft)
* **JDK:** Java Development Kit (JDK) 11 or higher

## 🚀 How to Run the Code
You can run these files directly through the VS Code "Run" button, or manually via the terminal using the following steps:

1.  **Open your terminal** (or use the integrated terminal in VS Code: `Ctrl + ``).
2.  **Compile the Java file:**
    ```bash
    javac FileName.java
    ```
3.  **Execute the program:**
    ```bash
    java FileName
    ```
